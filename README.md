# project_in_python
YouTube Video Manager (Python CLI Application)

Overview
YouTube Video Manager is a simple command-line application built using Python. It allows users to
manage YouTube video records efficiently using CRUD operations (Create, Read, Update, Delete).
The application stores video data such as name and duration inside a JSON file for persistent
storage.


Features
• View all saved YouTube videos
• Add a new video
• Update existing video details
• Delete a video
• Persistent storage using JSON file
• Interactive command-line interface


Tech Stack
• Python 3.10+
• JSON for data storage
• File Handling
• Functions and Loops
• Exception Handling
• Match-Case (Structural Pattern Matching)


How It Works
1. The program loads video data from youtube.txt file. 2. If the file does not exist, an empty list is
initialized. 3. Users choose an option from the menu to perform operations. 4. After each
modification, data is saved back to the JSON file. 5. The program continues running until the user
selects Exit.
